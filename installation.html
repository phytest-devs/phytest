InstallationÔÉÅ

Install Phytest using pip:

pip install phytest

Note

Requires Python >=3.8 & <3.11

For development clone the main brach from GitHub and install with poetry:

git clone https://github.com/phytest-devs/phytest.git && cd phytest
poetry install
poetry run pytest
